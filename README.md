# AI Content Evaluator (LangGraph)

## Overview
This project demonstrates a simple AI workflow using LangGraph to simulate content evaluation.

## Flow
User Input → Retrieval → Filtering → Risk Check → Ranking → Output

## Features
- Simulated RAG pipeline
- Content filtering (quality control)
- Risk/governance layer for evaluation
- Ranking based on engagement signals
- Modular node-based architecture

## Why This Matters
This reflects how AI systems:
- Improve content quality
- Apply governance logic
- Enhance user-facing outputs

## Tech Stack
- Python
- LangGraph
- Google Colab
